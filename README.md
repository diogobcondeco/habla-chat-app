# habla-chat-app

## Run Backend
```bash
$ cd backend
$ go run main.go
```
## Run Frontend
```bash
$ cd frontend
$ npm install
$ npm run dev
```
## Backend created with
```bash
$ cd backend
$ GO111MODULE=on
$ go mod init github.com/diogothedev/habla-chat-app
```
## Frontend created with
```bash
$ cd frontend
$ npm init vite@latest . -- --template react
```
